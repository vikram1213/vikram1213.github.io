# vikram-portfolio
This is a GitHub-ready static portfolio for Vikram Pathak.

## How to deploy (quick)
1. Create a new GitHub repository named `vikram-portfolio`.
2. Upload the contents of this ZIP (index.html and this README).
3. On GitHub: Settings → Pages → Deploy from branch `main` (or root). The site will be available at:
   `https://<your-username>.github.io/vikram-portfolio`
